1. Copy the folder "VideoGenerator".
2. Paste it somewhere out side the tacticsboard app.
3. Run the command "npm i" to download all the dependencies.
4. run "nodemon" to start the server.
5. Go to the tactics-board animations , add some objects on different frames.
6. Click the download button, the video will be downloaded after some time.
7. In the tacticsboard app and in node js index file, there is an instance named api, provide the url of your own loacalhost there on which node is running for example  (i.e localhost:8000).
Note:Node server will run with the node js version 16 or above.
